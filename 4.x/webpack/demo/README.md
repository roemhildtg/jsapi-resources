# Sample using @arcgis/webpack-plugin

## Usage

```
npm install

# run local dev server
npm start

# create a production build
npm run build
```

## Steps to get to bug

Run on command line:

```
npm i
npm start
```

Watch the app work!

Now kill the task with CTRL+C

Then run 

```
npm start
```

Watch the app fail! Open the console to make sure
